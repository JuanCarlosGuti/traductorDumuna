# Frontend — Corpus Damana

Pendiente para la Fase 3: aplicación Angular 18+ con standalone components
y señales.

Cuando se inicialice (`ng new`), hay que añadir `"frontend"` al array
`workspaces` del `package.json` raíz. No se incluye todavía porque npm
falla en `npm install` si un workspace listado no tiene `package.json`.
