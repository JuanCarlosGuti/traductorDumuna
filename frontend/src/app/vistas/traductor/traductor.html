Falta configurar un motor de traducción

Hay dos opciones; en ambos casos se definen variables de entorno y se reinicia el backend:

Opción A — Gratis, local (Ollama)

  1. Instala Ollama y ejecuta ollama pull qwen2.5:7b.
  2. Define en PowerShell: 
    setx TRADUCTOR_BASE_URL "http://localhost:11434/v1"
setx TRADUCTOR_MODELO "qwen2.5:7b"
setx TRADUCTOR_TIMEOUT_MS "300000"

También sirve cualquier API compatible con OpenAI: Google Gemini (clave gratis en aistudio.google.com), Groq, Hugging Face… definiendo además TRADUCTOR_API_KEY.

Opción B — Claude (mejor calidad, de pago)

  1. Crea una clave en console.anthropic.com (~US$0.01 por traducción).
  2. setx ANTHROPIC_API_KEY "sk-ant-..."

Después: abre una terminal nueva, reinicia el backend (npm start) y recarga esta página.

El resto de la aplicación funciona normalmente sin motor.
